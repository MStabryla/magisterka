# PL - Projekt magisterski z Informatyki Stosowanej

Projekt magisterski polegał na utworzeniu sieci robotów poruszających się w sposób autonomiczny. Robot miał za cel poruszać się do przodu i na podstawie pomiaru czujników odległości wykonywać skręt w celu uniknięcia przeszkody. Głównym celem utworzenia robotów było utworzenie sieci WiFi poprzez rozprowadzenie sygnało przez roboty w sieci Mesh.

## Technologie

Do utworzenia robota wykorzystano szablonowe komponenty elektroniczne jak czujniki odległości, moduł WiFi i baterie. Robot został zaprogramowany w języku C++ z wykorzystaniem platformy Arduino.

## Status

:white_check_mark: Zakończony

Projekt został zakończony we wrześniu 2023 roku.

# EN - Master's Project in Applied Computer Science

The master's project involved creating a network of autonomously moving robots. The robot's goal was to move forward and, based on distance sensors, turn to avoid obstacles. The primary goal of creating the robots was to create a WiFi network by distributing signals through the robots in a mesh network.

## Technologies

The robot was created using standard electronic components such as distance sensors, a WiFi module, and batteries. The robot was programmed in C++ using the Arduino platform.

## Status

:white_check_mark: Completed

The project was completed in September 2023.
